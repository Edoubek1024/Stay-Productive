# Stay Productive

Stay Productive helps you reclaim your time by blocking distracting websites and removing time-wasting features like infinite scrolling and recommended content. You can choose which platforms to filter (like YouTube, TikTok, Instagram, or Twitter) and even add your own custom sites. Stay focused with flexible, user-controlled productivity tools built right into your browser.
